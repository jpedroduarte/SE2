Detailed Description

Drivers for SPI on LPC2106.

Function SPI_Init must be called before anything else.

In this module you can find functionality to transfer data with SPI protocol. Using: P0.4 SCK P0.6 MOSI P0.7 SSEL For communication with LCD Nokia 6610.
