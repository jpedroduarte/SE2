Detailed Description

Drivers for LCD on LPC2106.

Function LCD_Init must be called before anything else. This module provides functionality to write characters defined in LCD_Font.c in whatever line/column desired.

Configuration: BackLight -> P0.1 RESET -> P0.3 LCD_CS -> P0.5
